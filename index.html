AI Calendar Assistant

An end-to-end assistant that manages my calendar through natural-language Telegram messages — deployed, running under systemd, and debugged in production.

Built with Flask and the Anthropic API (Haiku), integrated with Google Calendar via OAuth. Deployed on an Oracle Cloud free-tier VM (Ubuntu 22.04) under systemd, with Tailscale for secure remote access. The fun part was the real production debugging: OAuth token expiry, Node version conflicts, and switching providers to beat rate-limiting.

Mod Engineering

Deep technical work on a community character mod — patching compiled C# assemblies at the IL level and hunting down a subtle multiplayer desync.

Fixed localization by patching hardcoded string literals at the IL level inside compiled C# assemblies using dnSpyEx — no source access. Then diagnosed a multiplayer desync caused by non-deterministic RNG consumption flowing through a non-networked field, and reported the findings back to the mod authors to fold upstream.
